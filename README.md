# Game
因为上线需要软著有点麻烦，所以只上传了体验版。可微信扫描二维码玩一玩，申请体验，我可以在后台操作。

![photo](https://github.com/KaryKim/Game/blob/master/game.jpg)      
游戏是独立制作。     
有骨骼动画、碰撞、预制资源、计时器、对象池等   

其中位图字体用的Littera(http://kvazars.com/littera/)（良心推荐）      
此外因为源码包超出4M,采用了分包加载       
图片等素材用的外网资源
