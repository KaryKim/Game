# 角色的两种状态，得分或死亡
![photo](https://github.com/KaryKim/Game/blob/master/scene2.png)  



玩法：手动触屏操作会飞的龙，躲开弓箭的攻击，获取龙珠 

     倘若被弓箭所击，游戏结束 
     
     玩家通过不断获取龙珠数来得分 
     
     龙珠弹性碰撞并且会随机消失
